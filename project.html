Design and Implementation of IoT-BasedSCADA System

To overcome the limitations of conventional unidirectional data monitoring and control systems that are currently being used in industrial and commercial institutions, we would like to develop an Internet of Things (IoT) based monitoring and control system and a smart grid model. Using web server real-time data monitoring and optimizing from any location in the world. Internet things or IoT is a system of reticular computing devices, which is capable to transfer data information through the network while requiring system human-human or human-computer interaction. Merely it’s a networking system that internet-connected objects able to collect information, stored information, interchange information, and sent the web-server so that users can easily access the information any location in the world. Moreover, using IoT based SCADA system authorized to get the opportunity to optimize 24hours monitoring parameters facility and this system operates fully automatic.

Design a Vertical Climbing Window Cleaner Robot with Emergency Firefighting Support

• Designed a Window Cleaner Robot to climb any given vertical surface with the help of the Reverse Propulsion technology. The cleaning system was made as a versatile system that can be swapped for emergency firefighting support. Therefore, Reduced 90% of the human risk factor and the overall cost associated with cleaning skyscrapers.

Design a simulation-based Analog transconductance amplifier.

• Designed and simulated a transconductance amplifier under a 50dB low-frequency gain, -3dB bandwidth, and 5MHz Unity gain frequency in GFUS 130nm process using Silvaco.

• Applied a CMOS IC design and used PFET3.3 and NFET3.3 to an amplifier to convert an input voltage (VDD-3.3V) into an output current (60uA).
